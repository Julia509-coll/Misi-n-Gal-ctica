# 🚀 Proyecto: Sistema de Entrenamiento de Tripulación (Python)

Este repositorio contiene los retos de programación para el sistema de control de una misión espacial. El objetivo es practicar estructuras de control, bucles y manejo de errores en Python.

## 📋 Descripción del Proyecto
El sistema se encarga de monitorizar niveles críticos (como el combustible) y ejecutar protocolos de "autoreparación" mediante bucles de reintento.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Python 3.x
* **Entorno:** Jupyter Notebook / IPython Display (para visualización HTML)
* **Plataforma:** GitHub para control de versiones

## 🕹️ Cómo Funciona
El código principal incluye:
1. **Módulo de Diagnóstico:** Una función que evalúa niveles y devuelve estados (Óptimo, Precaución, Crítico).
2. **Sistema de Check Automatizado:** Un bucle que detecta fallos y aumenta los niveles automáticamente hasta que el sistema sea seguro.

## 🚀 Ejecución
Para probar el código, abre el archivo en un entorno que soporte Python o Jupyter y ejecuta las celdas de entrenamiento.
